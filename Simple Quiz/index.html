A Simple Quiz Suing javascript and Chrome Console

Open the Chrome Console using

  • Windows: Ctrl + Shift + I
  • Mac: Cmd + Shift + J
Please open up the Chrome Developer Console to take the Quiz.

The Questions will pop up randomly, so they may be displayed multiple times.

Score for answer to each question displayed is 1, otherwise 0.

Type 'exit' in the prompt to exit the game.
